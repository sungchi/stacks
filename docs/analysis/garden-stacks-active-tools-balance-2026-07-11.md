# Stacks 능동형 정원 도구 초기 밸런스 검증

> 실행일: 2026-07-11
> 대상 규칙: `active-garden-tools-v1`
> 시드: `7100~7119` 20개
> 상한: 런마다 생존 턴 100

## 방법

- `도구 미사용`: 내장 자동 추천으로 카드를 놓고, 합법 카드가 없을 때만 전체 손패를 교체한다. 도구 보상은 건너뛴다.
- `비상시 도구`: 같은 자동 추천을 사용하되 합법 카드가 없을 때 가지치기, 접붙이기, 옮겨심기 순으로 합법 대상을 찾는다.
- 두 전략은 같은 시드와 40장 순수 숫자 덱을 사용한다.

## 결과

| 전략 | 100턴 생존 | 평균 턴 | 평균 점수 | 평균 수확 | 도구 사용 |
|---|---:|---:|---:|---:|---:|
| 도구 미사용 | 20/20 | 100.0 | 8,174.1 | 19.30 | 0 |
| 비상시 도구 | 20/20 | 100.0 | 8,174.1 | 19.30 | 0 |

- 최소·최대 생존 턴은 두 전략 모두 100턴이다.
- 비상시 도구 전략에도 100턴 안에 일반 합법 배치가 끊기는 상황이 없어서 도구가 소모되지 않았다.
- 도구별 사용 편중은 사용 표본이 0이라 판단할 수 없다.

## 해석

능동형 도구 분리는 카드 정보량과 선택 구조를 개선하지만, 현재 자동 배치·정원 용량·교체 3회의 조합은 첫 100턴 생존 압력이 매우 낮다. 따라서 도구가 실수 복구나 장기 계획 수단으로 존재해도 비상 자원으로서의 필요성은 이 구간에서 드러나지 않는다.

이번 구현에서는 승인된 요구사항의 수치를 임의로 바꾸지 않는다. 다음 밸런스 실험은 도구 지급 주기보다 먼저 시작 교체 `3 -> 2`, 정원 용량 `10 -> 8~9`, 또는 높이 균형 제한 강화 중 하나만 바꾸고 동일한 20개 시드로 비교해야 한다. 목표는 도구 미사용 100턴 생존률을 낮추면서 도구 사용 전략이 생존률 또는 점수에서 분명한 차이를 만드는 것이다.

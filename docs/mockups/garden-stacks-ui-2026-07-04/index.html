현재 기본 UI 스타일이다. 흰 바탕, 검은 선, 회색 보조면, 컬러 게임 애셋을 기준으로 구현한다.

 P0

모바일 플레이

낮은 상태 스트립, 2x2 더미, 손패 근처 추천 한 줄, 하단 손패 레일을 확인한다.

 P1

데스크톱 플레이

테이블 중심 레이아웃, 선택 카드 상세, 약한 추천 버튼의 균형을 확인한다.

 P3

상점

구매 가능 항목, 비용, 구매 후 변화량이 바로 읽히는 모달 구조를 확인한다.

 P3

도감/결과

다음 발견 목표, 진행 칩, 보상과 결과의 다음 행동 중심 구성을 확인한다.
